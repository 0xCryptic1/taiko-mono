# Setup your wallet

This guide helps you connect your wallet to Taiko's networks.

## Prerequisites

- You have a wallet installed and available to your browser.

## Steps

<Steps>

    ### Add the Sepolia testnet

    <ConnectToMetamaskButton network="Sepolia" />

    ### Add the Taiko testnet (Grimsvotn)

    <ConnectToMetamaskButton network="Grimsvotn" />

    ### Add tokens to your wallet
    <AddTokenButton token="TTKO" network="Sepolia"/>
    <AddTokenButton token="BLL" network="Sepolia"/>
    <AddTokenButton token="HORSE" network="Sepolia"/>

    
    <AddTokenButton token="TTKO" network="Grimsvotn"/>
    <AddTokenButton token="BLL" network="Grimsvotn"/>
    <AddTokenButton token="HORSE" network="Grimsvotn"/>

</Steps>

## Troubleshooting

### The add chain buttons did not work

Try to manually add the network to your wallet using the [RPC configuration](/docs/reference/rpc-configuration), or by using [chainlist.org](https://chainlist.org) (make sure to check the box for "Include testnets").
