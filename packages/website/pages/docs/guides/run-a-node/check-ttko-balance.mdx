# Check TTKO/TTKOe balance

This guide is for checking your TTKO/TTKOe balance on the TaikoL1 contract.

## Steps

<Steps>

### View TTKO/TTKOe balance on TaikoL1 contract

Go to the TaikoL1 contract read page ([TTKO](https://sepolia.etherscan.io/address/0x6375394335f34848b850114b66A49D6F47f2cdA8#readProxyContract#F10)/[TTKOe](https://explorer.test.taiko.xyz/address/0x4e7c942D51d977459108bA497FDc71ae0Fc54a00/read-proxy#address-tabs)) and input your proposer or prover address to lookup your TTKO/TTKOe balance.

</Steps>
